Flex Layout

{ Properties }

  • display

    Defines the container as a flex container or grid.

    Specifies flex container behavior, enabling flexbox layout for children.

  • flex-direction

    Defines the direction of the flex items in the container.

    Determines whether items are arranged horizontally or vertically.

  • flex-wrap

    Allows flex items to wrap onto multiple lines if necessary.

    Controls wrapping behavior of flex items in a container.

  • justify-content

    Aligns flex items along the main axis (horizontal by default).

    Controls the alignment of items along the main axis.

  • align-items

    Aligns flex items along the cross axis (vertical by default).

    Defines how items align relative to the containerâ€™s cross axis.

  • align-self

    Allows individual flex items to override container's align-items.

    Enables a specific item to adjust its alignment independently.

  • flex-grow

    Defines how much a flex item should grow relative to others.

    Determines the growth factor of items within the container.

  • flex-shrink

    Defines how much a flex item should shrink relative to others.

    Controls how items shrink when container is smaller than items.

  • flex-basis

    Specifies the initial size of a flex item before any growing or shrinking.

    Sets the initial size of items before space distribution.

  • flex

    A shorthand for flex-grow, flex-shrink, and flex-basis properties.

    Combines grow, shrink, and basis into one convenient declaration.

  • order

    Defines the order of flex items within the container.

    Alters the visual order of flex items within the layout.

  • align-content

    Aligns flex lines when there is extra space in the cross axis.

    Distributes space between multiple flex lines across the container.
